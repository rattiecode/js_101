Sorting Arrays and Displaying Tables

The map() method is an iterative method. It calls a provided function once for each element in an array and constructs a new array from the results. This can be used to construct HTML code to display the parameters of an array of objects.

The map() method performs the function it is passed on each index of an array, and returns a new array with those changed values.

The join() method of an array creates a string of the values in the array, using the string passed into its function argument as the joiner between strings.
